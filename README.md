# Faster RCNN FPN v2


Improved Faster RCNN FPN v2, based on the paper [Benchmarking Detection Transfer Learning with Vision Transformers](https://arxiv.org/pdf/2111.11429)


The model was re-trained for experimentation and learning on the dataset downloaded from [Kaggle](https://www.kaggle.com/datasets/sixhky/open-images-bus-trucks).
